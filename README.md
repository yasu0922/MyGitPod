# MyGitPod
GitPod用リポジトリ

# 使い方
以下のURLにアクセスする

https://gitpod.io/#github.com/yasu0922/MyGitPod

# Jupyterlabを立ち上げる
以下のコマンドを実行して、Pipfileを使用してJupyterlabを起動する。

~~~
$ pipenv install
$ pipenv shell
([repository_name])$ jupyter lab
~~~
